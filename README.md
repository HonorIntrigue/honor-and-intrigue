# Honor + Intrigue

A system implementation of **[Honor + Intrigue](https://www.drivethrurpg.com/en/product/99286/honor-intrigue)**, a
swashbuckling RPG by Chris Rutkowsky, for FoundryVTT.

_Honor + Intrigue_ has no gaming license, but permission was granted to make this system.

Full content and PDFs are not included in this system. Support the author buy purchasing _Honor + Intrigue_
on [itch.io](https://basicactiongames.itch.io/honor-intrigue)
or [DriveThruRPG](https://www.drivethrurpg.com/en/product/99286/honor-intrigue), and consider purchasing the [_Tome of
Intriguing Options_](https://basicactiongames.itch.io/tome-of-intriguing-options) supplement as well!

## Features

### Compendiums

The system provides drag-and-drop compendium entries for items, careers, boons, flaws, and more, from the core rulebook
as well as the Tome of Intriguing Options.

Some macros are also provided for awarding GM Fortune and Advancement Points.

### Dueling Styles

Easily track progress of maneuver advancement for different Dueling Styles and choose the active style in an instant.

### Maneuver Resolution

Maneuvers rolled from character sheets include outcome resolution, quickly indicating degrees of success and enabling
players and GMs to adjust the outcome with Fortune or using Reactions.

### Ship Vehicles

Create actor sheets for ships and assign crew duties, boons/flaws, and more.

### Variant Rules

Support is provided for the d10 variant rule, with support for more variant rules such as polyhedral damage planned.
